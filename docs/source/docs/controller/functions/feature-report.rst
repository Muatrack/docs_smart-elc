数据报表
=========

    数据报表对终端采集到的数据进行了汇总和可视化。内容有 *用电量统计* 、 *当日实时用电数据* 两个大模块。
        
        * 用电量统计_ : 此模块将站点终端设备的历史用电量以图形方式呈现，其中集成了当日、当月、当年的用电统计。
        * 当日实时用电数据_: 默认展示终端在当天00:00 起始的实时 电流、电压、功率等数据曲线。

    数据报表入口: 

    *登录平台(运维账号) ->  站点设备信息 ->  目标站点(单击查询，选择站点) -> 数据报表*
